= Spring AI application with Spring Data Neo4j

This is an example application using Spring AI, Spring Data Neo4j, and Goodreads book data.

== Data set

Data is maintained and pulled from a public data source (more info coming soon!). An example of the data model is shown below.

image::src/main/resources/goodreads-data-model.png[]

For this project, we are focusing on the Book and Review entities and the relationship between them.
